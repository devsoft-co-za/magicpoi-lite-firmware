Constructs an instance of the ColourPatterns class.

This constructor initializes an instance of the ColourPatterns class with the specified pins for the red, green, and blue LEDs. It also sets initial values for various class variables used in LED patterns.

Parameters
redPinThe pin number connected to the red LED.
greenPinThe pin number connected to the green LED.
bluePinThe pin number connected to the blue LED.
Note
The constructor initializes the pins as OUTPUT and sets initial values for class variables such as previousMillis, rainbowWay, threeWay, ledState, upDownFade, fadeSpeed, and interval.
You can customize these variables to control the behavior of LED patterns.

void ColourPatterns::changeColours ( int  choice)

Changes the color pattern of RGB LEDs based on the provided choice.

This function allows you to change the color pattern of RGB LEDs based on a numeric choice. It uses a switch statement to select a specific color pattern corresponding to the choice and calls the respective color-setting function.

Parameters
choiceAn integer representing the chosen color pattern (0-13).
Note
The available color patterns are specified by the case values.
Default behavior is to turn off all LEDs.

void ColourPatterns::runLoading ( )

Runs a loading animation with RGB LEDs.

This function runs a loading animation by sequentially lighting up the RGB LEDs in red, green, and blue colors. It sets the red, green, and blue pins to different brightness levels to create the loading effect.

Note
This function is designed for RGB LED control and creates a visual loading effect by cycling through the primary colors.
The loading animation consists of three phases: red, green, and blue, each lasting for 500 milliseconds.
You can customize the timing and colors of the loading animation as needed.
